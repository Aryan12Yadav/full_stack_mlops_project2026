# Full Stack MLOps Project 2026

This repository contains the codebase for the Full Stack MLOps project.

## Project Structure

- `requirements.txt`: Tracks Python packages and dependencies.
- `.gitignore`: Standard gitignore file to exclude unnecessary files (like virtual environments, pycache, etc.).
- `vehicle1/`: Local Python virtual environment (ignored by Git).

## Setup Instructions

1. **Create the virtual environment** (if not already created):
   ```bash
   python3 -m venv vehicle1
   ```

2. **Activate the virtual environment**:
   - On Linux/macOS:
     ```bash
     source vehicle1/bin/activate
     ```
   - On Windows:
     ```cmd
     vehicle1\Scripts\activate
     ```

3. **Install the dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
